# Daily Note — Example Structure

Two formats are used depending on the day type. The weekly report parser expects consistent structure — adapt the sections to your needs but keep the headings stable.

---

## Weekday example

# 2026-01-01 Keskiviikko

### 1. Priorities
	- [ ] Priority 1
	- [ ] Priority 2
	- [ ] Priority 3

### 2. Progress
- [your tracked habit or activity]: [value or description]
- Success criteria: - [x] [your own criteria] — Kyllä/Osittain/Ei

### 3. What one thing would move the day forward most?
Free text here.

### 4. Tracked metrics
- [Metric 1] (1-5): 3
- [Metric 2] (hours): 7
- [Metric 3] (1-5): 2

### 5. Reflection
How did the day go? What worked, what did not?

### 6. Notes
Free observations.
