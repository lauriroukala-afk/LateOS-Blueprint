# Results

Saved results from the daily digest for further review.

Top-scoring results from the daily digest pipeline are stored here. The agent can help compare, prioritize, or follow up on items in this folder.

---

**Note on folder name:** This folder is named `JobOpportunities/` because the original system was built for a job search digest. Rename it to match your own use case — `Leads/`, `Articles/`, `Opportunities/`, or whatever fits what your digest surfaces.
